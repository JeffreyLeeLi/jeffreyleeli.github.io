To make windows desktop friendly to our eyes, we can set window color to `bean green`.

## Open `Personalize`

## Choose `Window Color`

## Open `Advanced Appearance Settings`

## Select `Window` under `Item`

## Choose `Others` in color drop down

## Set `HSV`

H: 85
S: 123
V: 205

## Add to `Customize colors`

## Click to use it

Done.
